第一章

HTMLの復習でdev タグとスタイルの関係を記述しています

style で　セレクタ名{ プロパティ:値; } と定義すると記述方法を定義できます。
body　で　dev class="XXXX" と記述すると
style で　dev.XXXX{ プロパティ:値 }でスタイルを定義できます。

第二章

body　で　dev id="XXXX" とすると
style で　#XXXX{ プロパティ:値 }でスタイルを定義できます。

id 指定　#XXXX
class 指定　.XXXX
どちらでもスタイルを指定できますが
id は同一ページ内で１度しか使用できません。
同一の名前がある場合は、id が優先されます。

画像などはid 指定にしましょう
